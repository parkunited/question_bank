# 문제은행(question_bank)
```

* '스프링부트'를 이용하여 만든 '문제은행'입니다.

* 문제를 직접 생성, 변경, 삭제, 출력 할 수 있습니다.

* 'JAVA'와 'Springboot'를 중점으로  MySql, HTML, Bootstrap, Thymeleaf 등을 개발도구로 활용하였습니다. 

* '프론트엔드'의 위치는 'question_bank/src/main/resources/'이며, '백엔드'의 위치는 'question_bank/src/main/java/com/ezen/question_bank/'입니다.

```
